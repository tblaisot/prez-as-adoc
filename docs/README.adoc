== Welcome

Welcome to the *prez-as-adoc* documentation! This comprehensive guide will help you understand and use this powerful tool for creating interactive presentations from AsciiDoc documents.

== What is prez-as-adoc?

*prez-as-adoc* is a tool that combines:

* *AsciiDoc* - Clean, semantic markup language
* *Bespoke.js* - Modern, flexible presentation framework
* *Vite* - Fast development and build tooling

The result is a workflow that lets you write presentations in simple text format while generating beautiful, interactive slide decks.

== Key Features Explained

=== Write in AsciiDoc

Simple, readable markup:

[source,asciidoc]
----
= My Presentation

== First Slide

* Point one
* Point two

== Second Slide

More content here.
----

=== Custom Templates

Create reusable slide layouts:

[source,html]
----
<div class="two-column">
    <div class="left">
        <!-- slot=left -->
    </div>
    <div class="right">
        <!-- slot=right -->
    </div>
</div>
----

=== Speaker Notes

Hidden notes for presenters:

[source,asciidoc]
----
== Slide Title

Visible content.

[.notes]
****
Private notes here.
****
----
